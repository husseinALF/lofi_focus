# 🎧 Lofi Focus Station

A modern, responsive, and premium web application designed for maximum productivity and relaxation. Built with blazing-fast React, TailwindCSS, and infused with luxurious micro-interactions using Framer Motion.

## ✨ Features

- **📺 Integrated YouTube Player:** Paste any YouTube link to stream your own music. Fully responsive and minimizable layout!
- **🍅 Pomodoro Timer:** Built-in timer following the scientific standard (25 minutes focus / 5 minutes break) that automatically logs and saves your daily "session stats".
- **🌧️ Ambient Sound Mixer:** Mix built-in background sounds (Rain, Fireplace, Keyboard typing, etc.) to craft your perfect audio landscape.
- **💾 Save Your Vibe:** Create your ultimate combination of sounds + visual theme and save the mix to your Dashboard with a custom name. Watch them smoothly slide in and out with satisfying animations!
- **🎬 Cinematic Background Themes:** Choose between a forest, cafe, or bedroom vibe, OR...
- **📺 Custom Video Background:** Paste a custom YouTube video link into the "TV icon" and it will play silently on a high-quality loop as your personalized video background.
- **✅ Framer Motion Tasks:** A smooth "Focus Task" to-do list featuring sleek cascade and slide animations when adding or completing tasks.
- **🎹 Global Hotkeys:** Keep your hands on the keyboard:
  - `Space` = Play / Pause music
  - `M` = Mute / Unmute all audio
  - `1`, `2`, `3` = Switch Themes
  - `H` = Open keyboard shortcuts menu

## 🚀 Tech Stack

- **React (`19.x`)** powered by Vite
- **TailwindCSS (`4.x`)** – for all styling and the immersive "glassmorphism" UI.
- **Framer Motion** – 60fps micro-interactions (cascading entries, layout transitions).
- **Lucide React** – Super clean iconography.
- **React YouTube** – Seamless custom video backgrounds and integrated music player.

## 💻 Getting Started

1. Ensure you have `Node.js` installed.
2. Clone or download this repository.
3. Run `npm install` in the project directory to fetch all modern dependencies.
4. Run `npm run dev` and open the provided localhost link.
5. Put your headphones on! 🎧

## 🎨 UI/UX Design Philosophy

The entire application is built as a strict Single Page Application (SPA). The interface elegantly adapts and never stretches beyond the viewport height, entirely eliminating unwanted vertical scrolling. Embodying absolute premium 2026 design concepts, it delivers a true "Dashboard" feel right in your browser.
